Package: groovyx.caelyf.logging

[Groovy] Class GroovyLogger

    • groovyx.caelyf.logging.GroovyLogger

    • Simple wrapper for java.util.logging.Logger. This logger can log correct class/method names even if used in Groovy.

      Implementation idea found here.

      Authors:
      ksky@jggug.org (original implementation)
      Guillaume Laforge (minor groovyfications and comments)
