Package groovyx.caelyf

  • Class Summary 
    Class Description
    BindingEnhancer Class responsible for adding adding various services and variables into the binding of Groovlets and Templates.
    BindingsTransformation This Groovy AST Transformation is a local transformation which is triggered by the Groovy compiler when developers annotate their classes with the @CaelyfBindings annotation.
    CaelyfCategory @author Guillaume Laforge
    CaelyfServlet The Caelyf servlet extends Groovy's own Groovy servlet
    CaelyfServletContextListener Servlet context listener configured on startup for calling the initialization routines of the plugin system.
    CaelyfTemplateServlet The Caelyf template servlet extends Groovy's own template servlet
    ExpirationTimeCategory Category methods for time duration handling, for the caching expiration definition
  • Annotation Type Summary 
    Annotation Type Description
    CaelyfBindings If you annotate your classes with the @CaelyfBindings annotation, the compiler will automatically inject the variables and services that are usually injected in your Groovlets and templates.
